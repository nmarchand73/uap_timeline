# DIRD — Defense Intelligence Reference Documents

Les PDF sont servis localement car l'accès distant (Black Vault) ne fonctionne pas.

## Téléchargement manuel

1. Téléchargez le ZIP (87 Mo) dans votre navigateur :
   **https://documents2.theblackvault.com/documents/dia/AAWSAP-DIRDs/AAWSAP-DIRDs-1-37.zip**

2. Enregistrez le fichier dans ce dossier : `public/dirds/AAWSAP-DIRDs-1-37.zip`

3. Exécutez :
   ```bash
   npm run download-dirds
   ```

Le script extrait les 37 PDF et met à jour `content/dirds.json` avec les chemins locaux.
